# Calculatrice simple en langage python

une calculatrice très simple d'utilisation 😁👍 pour vous procurer une expérience utilisateur fluide.

# Fonctionnalités
- Opérations : Addition, Soustraction, Multiplication, Division.
- Design : Interface sombre et minimaliste, réalisé uniquement avec tkinter
- Sécurité : Gestion des exceptions mathématiques et 0 bug en vue😏

# 🚀 Installation
1. si vous voulez télécharger le code vous pouvez cloner le depot avec le lien ci-dessous :
   `git clone https://github.com/ton-pseudo/Calculatrice-Python.git`
2. Lancez l'application :
   `python calculatrice.py`

# Créer l'exécutable
Si vous voulez générer le fichier .exe vous-même :
`pyinstaller --noconsole --onefile calculatrice.py`
